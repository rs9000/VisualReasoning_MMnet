# VisualReasoning_MMnet
Models in Pytorch for visual reasoning task on Clevr dataset. <br><br>
**Stack attention**:<br>
https://arxiv.org/pdf/1511.02274.pdf<br><br>
**Module network**:<br>
https://arxiv.org/pdf/1705.03633.pdf<br><br>
Yes, but what's new? <br>
